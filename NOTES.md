# Repository note

This repository preserves the 2022 public-facing CRUSH COVID dashboard code in archival form.

The original dashboard script was written by **Georgios Varotsis** within the framework of the **CRUSH COVID initiative**, a collaboration between **Region Uppsala** and **Uppsala University**.

This repository is intended for:
- historical reference
- code transparency
- reproducibility review of the public dashboard logic

Important context:
- the application reflects the original 2020-2022 implementation period
- the live dashboard is no longer maintained
- syndromic-surveillance indicator tabs that existed in earlier versions are no longer part of the public app
- any public release should include only disclosure-safe, aggregated data
